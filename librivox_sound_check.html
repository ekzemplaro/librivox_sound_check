sound_check for 1-Minute Test (May/07/2014)

Here you can check the technical specs of your recording.

Summary of tech specs

Channel: Mono
Sample rate: 44100 Hz
Sample format: 16 bit
Export format: MP3
Export bitrate: 128 kbps

Volume: 89 dB

1) First upload your recording in tests folder at LibriVox.

Or you can use a existing mp3 file in tests folder at LibriVox.
2) Enter your file name (Ex. test_ekzemplaro.mp3) here & submit.

This program fetch data from <https://librivox.org/uploads/tests/>.
